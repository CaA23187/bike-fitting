# 使用方法

使用前请先测量身体数据：根据**身体数据测量.jpg**计算跨高、臂长、胸高、身高。

参考输入：跨高79, 臂长60, 胸高124, 身高173

```shell
python self_Fitting.py
```

根据提示输入即可获得计算的车架参数和stack+, reach+参数。
